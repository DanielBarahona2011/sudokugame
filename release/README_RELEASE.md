# Sudoku Game — Release v1.0

Archivo: `Sudoku_Game.exe`

## Instalación

1. Descarga `Sudoku_Game.exe`
2. Ejecútalo con doble clic
3. No requiere instalación adicional

## Requisitos

- Windows 10/11 (64 bits)
- ~25 MB de espacio libre

## Contenido

- Sudoku con 3 dificultades
- Guardado en base de datos local
- Ranking de mejores tiempos
- Cronómetro, pistas y pausa

Para el código fuente y desarrollo, ver el [README principal](../README.md).
